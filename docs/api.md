# API 文档

## MarkdownConverter 类

### 构造函数

```javascript
new MarkdownConverter(options)
```

参数：
- `options.input`: 输入目录，默认 `'./docs'`
- `options.output`: 输出目录，默认 `'./dist'`

### 方法

#### convertAll()

转换所有 Markdown 文件。

```javascript
converter.convertAll();
```
